# afiliado-ml
**Cliente:** Dalberson<br>
<br>
**Projeto:**<br>
Consiste em um sistema de vendas para afiliado do Mercado Livre, em que o Administrador,
gerencia quais produtos ele é afiliado e ele irá colocar no seu Website; tendo assim
mais vendas e lucratividade.

**Fluxo de Compra:**<br>
Os usuários(finais) entram no website -> Veem os produtos cadastrados -> Clicam em *Comprar* ->
São redirecionados para a compra do produto no próprio mercado livre

**Perguntas do Faq: FaqView**<br>
